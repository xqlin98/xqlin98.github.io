Research Interests

I work on data-centric AI, with a focus on the following areas:

Data Selection & Attribution

  • Automatic data selection for LLM training
  • Data attribution and valuation

Prompt Optimization

  • Automatic prompt optimization for LLM inference
  • In-context learning and few-shot prompting

Others

  • Federated and collaborative machine learning
  • Zero-shot optimization
  • Preference-based optimization
